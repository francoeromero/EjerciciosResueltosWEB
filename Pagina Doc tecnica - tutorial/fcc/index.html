Intro

JavaScript es un lenguaje de programación de alto nivel, dinámico y orientado a objetos. Se utiliza principalmente en el desarrollo de páginas web interactivas y aplicaciones web. Es uno de los lenguajes de programación más populares en el mundo, con una amplia comunidad de desarrolladores y una gran cantidad de recursos en línea disponibles. JavaScript se ejecuta en el navegador del cliente, lo que significa que puede ser utilizado para crear aplicaciones web interactivas sin necesidad de acceder a un servidor.

JavaScript se integra fácilmente con otros lenguajes, como HTML y CSS, lo que facilita la creación de aplicaciones web complejas.

La comunidad de JavaScript está en constante evolución y mejora, lo que significa que hay un flujo constante de nuevas características y mejoras en el lenguaje.

En resumen, JavaScript es un lenguaje de programación versátil y potente que se puede utilizar para una amplia gama de aplicaciones web, desde páginas web simples hasta aplicaciones complejas. Con una amplia comunidad de desarrolladores y una gran cantidad de recursos disponibles, es una excelente opción para aquellos interesados en el desarrollo web.

Hay muchas razones por las que alguien puede elegir usar JavaScript:

  • Interacción en tiempo real: JavaScript permite crear páginas web interactivas y aplicaciones en línea que pueden responder a la entrada del usuario en tiempo real.

  • Mejora la experiencia del usuario: JavaScript puede ser utilizado para agregar efectos visuales, animaciones y otros elementos que mejoran la experiencia del usuario en una página web.

  • Accesibilidad: JavaScript permite crear aplicaciones accesibles para personas con discapacidades, lo que contribuye a una web más inclusiva.

  • Creación de aplicaciones en línea: Con JavaScript, es posible crear aplicaciones completas que se ejecutan en un navegador, lo que significa que no es necesario descargar o instalar software en el dispositivo del usuario.

  • Popularidad y amplia comunidad: JavaScript es uno de los lenguajes de programación más populares en el mundo, lo que significa que hay una gran cantidad de recursos y una amplia comunidad de desarrolladores disponibles para ayudar y resolver problemas.

Variables

En JavaScript, una variable es un espacio en la memoria que se utiliza para almacenar valores. Las variables se pueden usar para almacenar datos como números, cadenas de texto, objetos, arreglos, etc. Las variables se declaran con la palabra clave "var" o "let" o "const".

Con "var" se pueden re-declarar las variables y su alcance es global y dentro de funciones.

Con "let" y "const" se pueden re-asignar las variables pero su alcance es solo dentro del bloque en el que está definido. "const" además de ser limitado al bloque no se puede re-asignar.

LET

  • Permite re-asignar el valor de la variable.

  • Tiene un alcance de bloque, lo que significa que la variable es accesible solo dentro del bloque en el que se declaró.

CONST

  • No permite re-asignar el valor de la variable una vez que se ha asignado por primera vez.

  • Tiene un alcance de bloque, lo que significa que la variable es accesible solo dentro del bloque en el que se declaró.

VAR todavía se puede usar en JavaScript, pero no se recomienda su uso por varias razones:

  • Alcance de función: La variable declarada con "var" tiene un alcance de función, lo que significa que es accesible en toda la función, incluyendo bloques dentro de la función. Este comportamiento puede causar problemas de lógica y difíciles de depurar.

  • Re-declaración: "var" permite la re-declaración de variables con el mismo nombre, lo que puede causar problemas de lógica y difíciles de depurar.

  • Confusión con "let" y "const": "var" se comporta de manera diferente a "let" y "const", lo que puede causar confusión y errores fáciles de evitar.

let

nombre = "Juan";
console.log(nombre);

// imprime "Juan"



const

PORCENTAJE = 0.5;
console.log(PORCENTAJE);

// A la constante recomiendo que sea con mayúscula, son valores fijos



const

PI = 3.14;
console.log(PI);

// son valores que nunca cambian y que se las puede llamar pero no reasignarlas

JavaScript tiene seis tipos de datos primitivos:

  • String: Una secuencia de caracteres, por ejemplo: "Hola mundo".

  • Number: Un número, por ejemplo: 42 o 3.14.

  • Booleanos: Un valor lógico que puede ser verdadero o falso, por ejemplo: true o false.

  • Undefined: Un valor que indica que una variable ha sido declarada pero todavía no tiene un valor asignado

  • Null: Un valor que representa la ausencia intencional de cualquier valor

  • Symbol: Un tipo de datos único e inmutable

Estos son los tipos de datos primitivos de JavaScript y no son objetos. Algunos otros tipos de datos, como ArrayyObject , son objetos y no son primitivos.
